# EternalBlue (MS17-010) Vulnerability Assessment & Penetration Testing

## Overview
This repository contains a Vulnerability Assessment and Penetration Testing (VAPT) report on the MS17-010 (EternalBlue) vulnerability affecting Microsoft Windows systems via SMBv1.

The assessment was conducted in a controlled academic lab environment for learning and demonstration purposes.

## Scope
- Target OS: Windows 7
- Vulnerability: MS17-010 (CVE-2017-0144)
- Attack Type: Remote Code Execution
- Access Level: SYSTEM / Administrator

## Tools Used
- Kali Linux
- Nmap
- Metasploit Framework

## CVSS
- Base Score: 9.3 (Critical)

## Mitigation
- Apply Microsoft MS17-010 patch
- Disable SMBv1
- Network segmentation
- IDS/IPS monitoring

## Disclaimer
This project is strictly for educational purposes. No real-world systems were targeted.
